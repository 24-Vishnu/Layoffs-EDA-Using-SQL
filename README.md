**SQL Exploratory Data Analysis**

**Dataset Used: Layoffs Dataset**

This project focuses on exploring a layoffs dataset using SQL to uncover trends, patterns, and insights from the data. The analysis was performed on a cleaned dataset prepared during the data cleaning phase.

The goal of this project is to understand how layoffs were distributed across different companies, industries, countries, and time periods, while strengthening practical SQL skills through real-world data analysis.

**Project Files**

    01_Data_Exploration.sql
    02_Company_Analysis.sql
    03_Industry_Analysis.sql
    04_Time_Series_Analysis.sql
    
**Areas of Analysis**
1. Data Exploration

Reviewed the dataset structure, examined date ranges, and identified key characteristics such as the number of companies, industries, and countries represented in the data.

2. Company Analysis

Analyzed layoff trends across companies to identify organizations with the highest workforce reductions and compare the overall impact of layoffs.

3. Industry Analysis

Examined how different industries were affected and identified sectors that experienced significant workforce reductions.

4. Country Analysis

Explored layoffs across countries to understand regional trends and compare the impact of layoffs in different parts of the world.

5. Time Series Analysis

Analyzed layoff patterns over time to identify yearly and monthly trends and observe changes in workforce reductions across different periods.

**Skills Demonstrated**

* SQL Querying
* Aggregations (SUM, AVG, COUNT)
* GROUP BY
* ORDER BY
* CTEs
* Window Functions
* RANK() and DENSE_RANK()
* Time-Based Analysis

**Tools Used**

* MySQL

**Outcome**

Through this project, I gained hands-on experience in exploratory data analysis using SQL and developed a better understanding of how data can be analyzed to extract meaningful insights and support data-driven decision-making.
